# Research-Project
Worked on a research titled "Biases in Algorithms" which was later on published in JENER Journal of Empirical and Non-Empirical Research


In recent years, recommendation algorithms have become increasingly common on digital platforms. If a person is interested in something and they search for specific content, then that leads to more of the same content showing up in their recommendations. This study investigates, does a simple recommendation algorithm develop bias over time? The hypothesis was that recommendations depend on past user clicks; bias would increase over time. I created a python simulation with three content categories that were run for 50 cycles. The algorithm increased recommendation probability based on users clicks. One category became dominant while others decreased. Investigation concludes that simple algorithms can unintentionally create bias.
